## ‍💼 «Product»
Multi-page website for a business promotion company

## 🖥️ Deployment
To deploy this project run
```
$ npm install
$ npm run dev
```

## 🛠 Skills
- Pug.
- Sass (SCSS syntax).
- JavaScript (ES6).

## 🛠 Features
- Custom fonts.
- Light/dark theme switcher.
- Using WebP images.
